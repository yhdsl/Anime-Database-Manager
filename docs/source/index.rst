欢迎使用 ADM 项目 [1]_，这是针对重症仓鼠病爱好者开发的一个软件，\
其集成了本地数据库管理，网络通讯以及文件批处理功能。本软件目的在于缩短\
浪费在整理资源上的时间，使得文件整理变得便捷，同时不必为可以交给软件自动化\
事情而费心。

ADM项目立足于让欣赏变得简单和纯粹。

.. [1] 本项目全称为 Anime Database Manager，但为了简介起见，以下均简称为 ADM
