项目结构
--------

ADM项目是由如下三个部分构成的：

+ **core**:
    ADM项目的核心部分的实现，负责所有实际的代码逻辑，所有与网络和文件的直接\
    访问均由该部分实现

+ **website**:
    ADM项目的Web端的实现，负责所有网页的代码逻辑，主要负责网站相关的操作包装

+ **gui**:
    ADM项目的GUI界面的实现，负责所有UI的代码逻辑，与 **website** 的行为类似

此外ADM项目通过启动层初始化以上内容，并执行额外的初始化操作和最外层的异常捕捉。

    当前的 **website** 和 **gui** 部分的开发尚未开始，相关文档仅供参考
