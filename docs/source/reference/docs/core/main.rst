核心部分
==========

.. automodule:: core

本部分介绍了ADM项目的核心部分，其是项目的主体部分。

核心部分包含如下几部分

+ **底层层 (base)**: 负责提供所有的底层代码，以及全局可用的变量
+ **网络层 (net)**: 负责与互联网交互
+ **任务层 (task)**: 负责协调任务进行，以及内容通讯
+ **规则层 (rule)**: 负责库管理的具体逻辑的实现，是文件层等的操作封装
+ **文件层 (file)**: 负责实现各文件的具体操作
+ **接口层 (api)**: 负责提供可用的api，包括代码接口以及命令行支持，是对规则层的进一步封装
